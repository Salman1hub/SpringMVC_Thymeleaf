# SpringMVC_Thymeleaf
# 🌸 Spring MVC + Thymeleaf Demo

A simple web application built using **Spring MVC** and **Thymeleaf** template engine.  
This project demonstrates how to build a basic form handling flow — from displaying an input form to showing confirmation data.

---

## 🚀 Features
- 🧱 Built with **Spring MVC** architecture  
- 🌿 Integrated with **Thymeleaf** for dynamic HTML rendering  
- 🧾 Demonstrates form data binding using `@ModelAttribute`  
- ✅ Includes input validation using `@Valid` and `BindingResult`  
- 🎨 Basic CSS styling for better UI

## 🧠 How It Works

1. User opens the **form page** (`/showForm`).
2. Fills in student details.
3. Submits the form.
4. Data is processed by the controller and displayed on the **confirmation page**.
